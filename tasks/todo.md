# 手書き数字認識アプリケーション - プロジェクト計画

## 現在のプロジェクト状況
手書き数字認識のフルスタックWebアプリケーションプロジェクトです。React + Express + Python（scikit-learn）の構成でMNISTデータセットを使用した数字認識機能を実装しています。
