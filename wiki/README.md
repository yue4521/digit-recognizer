# GitHub Wiki用コンテンツ

このフォルダには、GitHub Wiki用に準備されたMarkdownファイルが含まれています。

## Wiki作成手順

1. GitHubのリポジトリページ（https://github.com/yue4521/digit-recognizer）にアクセス
2. "Wiki"タブをクリック
3. "Create the first page"をクリックしてWikiを初期化
4. 以下のファイルを順番にWikiページとして作成：

### 作成すべきページ

1. **Home** - `Home.md`の内容をコピー
2. **Installation** - `Installation.md`の内容をコピー
3. **API-Reference** - `API-Reference.md`の内容をコピー
4. **Architecture** - `Architecture.md`の内容をコピー
5. **Development-Guide** - `Development-Guide.md`の内容をコピー
6. **ML-Details** - `ML-Details.md`の内容をコピー
7. **Troubleshooting** - `Troubleshooting.md`の内容をコピー

## 注意事項

- ページタイトルはハイフン区切り（例：API-Reference）で作成
- すべてのファイルはdocs/フォルダの内容をベースにしています
- 内部リンクが適切に動作するようにページタイトルを統一してください
